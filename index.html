Era 01

Building the Toolbox

1999 – 2005

Simon Galbraith and Anthony Eden found Redgate to solve their own problem: comparing SQL Server schemas was painful and manual. SQL Compare becomes the category-defining tool that funds everything that follows.

🔍
SQL Compare
The product that started it all
Active

Schema comparison and synchronization for SQL Server. Built to scratch the founders' own itch, it became the category-defining tool used by DBAs worldwide — and still ships today.

SQL Server Schema Sync Founding Product
1999
2001
📊
SQL Data Compare
Schema's natural companion
Active

Compare and synchronize table data between SQL Server databases. Customer demand for SQL Compare naturally extended to the data inside — completing the comparison story.

SQL Server Data Sync

ANTS Performance Profiler
Redgate enters the .NET world
Active

Line-level performance profiling for .NET applications. Redgate's first move beyond SQL Server signalled a broader ambition: serve every developer who fights slow, unpredictable code.

.NET Performance First Expansion

SQL Packager

Strategic Pivot
c. 2005 – 2006

"One Tool Doesn't Tell the Whole Story"

Individual productivity tools were selling well, but customers kept asking for more of the workflow. Redgate's shift: stop shipping isolated utilities and start building a suite — every touchpoint from writing SQL to shipping it safely to production. The next era would expand the product line dramatically.

Era 02

The Developer Suite

2006 – 2010

Redgate expands from a handful of tools to a full suite. SQL Prompt transforms daily productivity; the .NET Reflector acquisition is the first major M&A; SQL Monitor enters real-time operations; and SQL Source Control plants the seed of everything that follows.

2006
✍️
SQL Prompt
Intelligent SQL autocomplete & formatting
Active

IntelliSense-style autocompletion, code formatting, and snippet management inside SSMS and VS. Became one of Redgate's most beloved tools — developers who tried it rarely returned to bare SSMS.

SQL Server IDE Productivity
💾
SQL Backup Pro
Compressed, encrypted SQL Server backups
Retired

Automated, compressed and encrypted backups with built-in verification. As SQL Server's native backup capabilities improved and cloud object storage matured, the niche gradually narrowed to retirement.

SQL Server Backup Operations
2007
2008
🔭
.NET Reflector
The legendary decompiler — acquired
Acquired

Acquired from creator Lutz Roeder, .NET Reflector was the de-facto .NET assembly browser and decompiler used by millions. Redgate's first major acquisition, proving M&A as a viable growth strategy.

.NET Decompiler First Acquisition
📡
SQL Monitor
Real-time SQL Server health monitoring
Evolved →

Real-time performance monitoring, alerting, and diagnostics for SQL Server estates. Later rebranded as Redgate Monitor when it expanded to PostgreSQL and Azure — the name change itself a signal of strategic direction.

SQL Server Monitoring → Redgate Monitor
2009
2010
🌿
SQL Source Control
Finally — databases in version control
Active

Version-control SQL Server schemas directly from SSMS. A landmark product that brought databases into the DevOps era — it planted the seed for the entire Database DevOps strategy that defined the next decade.

SQL Server Git DevOps Seed
Strategic Pivot
2011

"Databases Got Left Behind in DevOps"

The industry had embraced CI/CD for application code — but databases were still managed through risky, manual, Friday-night processes. Redgate's insight: be the company that completes the DevOps story by bringing the database change lifecycle into the same pipelines teams already used for everything else.

Era 03

The Database DevOps Era

2011 – 2018

Redgate plants its flag in Database DevOps — helping teams apply automated testing, CI/CD, and governance to the full database change lifecycle. A key acquisition fills the last missing piece.

🧪
SQL Test
Unit testing for SQL Server
Active

Unit testing framework for SQL Server databases, built on the open-source tSQLt. Automated testing was the missing link — without it, no one could confidently automate database deployments.

SQL Server Testing tSQLt
2011
2012
📈
DLM Dashboard
Database Lifecycle Management visibility
Evolved →

Track schema drift and enforce database change policies across your SQL Server estate. "Database Lifecycle Management" became Redgate's organising concept — the framework everything else plugged into.

SQL Server DLM Governance
📄
SQL Doc
Automated database documentation
Active

Automatically generate HTML documentation for SQL Server databases. As database complexity grew, keeping documentation manually up-to-date became untenable — SQL Doc automated the whole thing.

SQL Server Documentation
2013
2015
🚀
SQL Change Automation
ReadyRoll acquired — migrations-first CI/CD
Acquired

ReadyRoll was acquired and rebranded as SQL Change Automation — a migrations-based pipeline approach that integrated directly with Azure DevOps, TeamCity, and Jenkins, filling the final gap in automated database delivery.

Migrations CI/CD Azure DevOps Acquisition
🗂️
SQL Data Catalog
Classify and govern sensitive data
Active

Classify and document sensitive data (PII, financial records) across SQL Server databases — a direct response to GDPR and the growing enterprise demand for data governance visibility.

Data Governance GDPR Classification
2017
Strategic Pivot
2019

"Open Source Is a Feature, Not a Threat"

The acquisition of Flyway — the world's most popular open-source database migration tool with millions of users across dozens of databases — marked a fundamental philosophical shift. Redgate embraced open source as a growth channel, doubled its total addressable market overnight, and committed to a future that was genuinely cross-platform.

Era 04

Open Source & Multi-Platform

2019 – Present

Redgate broadens beyond SQL Server, embraces open source, and addresses the full data lifecycle — development, deployment, governance, compliance, and cloud. The portfolio grows from a SQL Server toolbox to a true multi-database DevOps platform.

2019
🦋
Flyway
World's most popular OSS migration tool — acquired
Acquired

Acquired from creator Axel Fontaine, Flyway had millions of users across PostgreSQL, MySQL, Oracle, SQL Server, and more. The acquisition doubled Redgate's addressable market and gave it a genuine open-source developer community to grow from.

Open Source Multi-platform Migrations Landmark Acquisition
🖥️
Redgate Monitor
SQL Monitor reborn as multi-platform
Active

SQL Monitor was rebranded and expanded to Redgate Monitor, adding PostgreSQL and Azure SQL support. The deliberate name change dropped "SQL" entirely — signalling that the company's future was not SQL Server-only.

SQL Server PostgreSQL Azure SQL Multi-platform
